# Physio Companion AI

This is a Streamlit web application that serves as a patient-facing AI companion embedded in a digital physiotherapy rehabilitation platform.

## Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone <your-repo-url>
   cd physio-chatbot
   ```

2. **Install requirements:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the app:**
   ```bash
   streamlit run app.py
   ```

## Deployment
You can easily deploy this repository on [Streamlit Community Cloud](https://streamlit.io/cloud). Connect your GitHub account, select this repository, and point it to `app.py`. The required packages are already listed in `requirements.txt`.
